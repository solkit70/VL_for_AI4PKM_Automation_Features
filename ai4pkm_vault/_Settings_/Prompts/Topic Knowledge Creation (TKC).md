Create new topic notes by scanning PKM for related content and establishing source-linked knowledge bases.

## Input
- Topic name/subject for new note creation
- [[Topic Template]] for structure
- PKM vault content for related learning and experience
- [[Topic Knowledge Addendum (TKA)]] guidelines

## Output
- New topic note following template structure
- Source-linked content items
- Related topic cross-references
- Organized knowledge collection on the topic

## Main Process
```
1. TEMPLATE SETUP
   - Create new note using [[Topic Template]]
   - Establish proper structure and sections
   - Set up metadata and properties

2. CONTENT DISCOVERY
   - Scan PKM for related learning and experience
   - Follow [[Topic Knowledge Addendum (TKA)]] guidelines
   - Identify relevant sources and materials

3. SOURCE LINKING & ORGANIZATION
   - Link all items to specific sources
   - Organize content thematically
   - Cross-reference related Topics
```

## Caveats
### Source Attribution Requirements
⚠️ **CRITICAL**: All items must be linked to a source

### Content Standards
- Don't add anything not attributable to a specific source
- Follow [[Topic Knowledge Addendum (TKA)]] guidelines
- Maintain one line per source format

### Cross-Referencing
- Link other related Topics in the vault
- Establish knowledge connections
- Validate all topic links before adding
