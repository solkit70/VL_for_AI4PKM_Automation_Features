## How to Publish

1. Create a PR with doc changes
   https://github.com/jykim/AI4PKM/pull/37
   
2. In Pages setting, point to the new branch & save
   https://github.com/jykim/AI4PKM/settings/pages
   ![[page_setting.png]]

3. In Actions page, you can check the progress of building the new website
   ![[page_building.png]]
